## La commande grep
La commande **grep** permet de **sélectionner des lignes** d'un fichier passé en argument qui respectent un motif donné en argument.
S'il n'y a pas de fichier en argument, grep lit sur l'entrée standard.
Grep peut donc être utilisé comme un filtre.

Sans options:  
```bash
grep MOTIF [fichier]
```

Quelques caractères spéciaux:
- `^`: début de ligne
- `$`: fin de ligne
- `.`: n'importe quel caractère
- `[...]`: un des caractères entre crochets
- `[^...]`: un caractère qui n'est pas entre crochets

### La répétition
- `c?`: 0 ou 1 occurrence de c
- `c*`: 0 ou plusieurs occurrences de c
- `c+`: 1 ou plusieurs occurrences de c
- `c{n}`: exactement n occurrences de c
- `c{m,n}`: entre m et n occurrences de c

`grep -E ""` permet d'utiliser ces caractères spéciaux.


### Les classes
- `[:alnum:]`: caractères alphanumériques
- `[:blank:]`: espace et tabulation
- `[:digit:]`: chiffres
- `[:lower:]`: lettres minuscules
- `[:upper:]`: lettres majuscules

### Les groupes
Un groupe est délimité par des parenthèses `()`. La ième occurrence d'un groupe peut être rappelée par `\i`.
ex:
```bash
$echo "(..).*([a-z][0-9]).*\2.*\1"
```

Déspécialisation:
On déspécialise un caractère avec  \  
ex:
```bash
$ls | grep -E "\.txt"
```

### "Ou" avec grep:
```bash
$grep -E -e "ab" -e "z$" fichier
```

### "Et" avec grep:
```bash
$grep -E "ab" fichier | grep -E "z$"
```
